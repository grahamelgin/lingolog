# LingoLog

A language learning tracker to log study sessions and visualize progress.

**Live Demo:** [https://lingolog-flame.vercel.app](https://lingolog-flame.vercel.app)

## Features
- Track multiple languages
- Log study sessions by category (Reading, Listening, Speaking, etc.)
- View progress with heatmaps and statistics
- Dark mode support

## Tech Stack
- Frontend: React + Vite (deployed on Vercel)
- Backend: Node.js + Express (deployed on Render)
- Database: PostgreSQL (Supabase)
